\# 🤖 Robot Line Follower Obstacle



🚀 Proyek ini merupakan implementasi robot cerdas berbasis \*\*Arduino Uno\*\* yang mampu \*\*mengikuti garis (line follower)\*\* sekaligus \*\*menghindari rintangan (obstacle avoidance)\*\* secara otomatis.



Robot ini dirancang sebagai bagian dari tugas mata kuliah Robotika pada Program Studi \*\*Teknologi Rekayasa Komputer\*\*.



\---



\## 🧠 Deskripsi Sistem



Robot bekerja dengan menggabungkan dua sistem utama:



\* \*\*Line Follower\*\* → mengikuti jalur garis secara otomatis

\* \*\*Obstacle Avoidance\*\* → mendeteksi dan menghindari rintangan di depan



Dengan kombinasi ini, robot mampu bergerak lebih adaptif dan cerdas di berbagai kondisi lintasan.



\---



\## ⚙️ Komponen yang Digunakan



\* 🔹 Arduino Uno

\* 🔹 Sensor Ultrasonik (HC-SR04)

\* 🔹 Motor Driver L298N

\* 🔹 Motor DC

\* 🔹 Servo Motor

\* 🔹 Sensor IR Line Follower

\* 🔹 Baterai \& Chassis Robot



\---



\## 🔧 Cara Kerja



1\. Sensor IR membaca garis pada lintasan

2\. Robot bergerak mengikuti jalur garis

3\. Sensor ultrasonik mendeteksi jarak objek di depan

4\. Jika ada rintangan:



&#x20;  \* Robot berhenti

&#x20;  \* Servo mengarahkan sensor untuk scanning

&#x20;  \* Robot mencari jalur aman

5\. Robot kembali mengikuti garis
