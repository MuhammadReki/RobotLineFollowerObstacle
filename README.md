# 🤖 Robot Line Follower Obstacle

🚀 Proyek ini merupakan implementasi robot cerdas berbasis Arduino Uno yang mampu mengikuti garis (line follower) sekaligus menghindari rintangan (obstacle avoidance).

## 🧠 Deskripsi

Robot ini mampu:

* Mengikuti garis otomatis
* Mendeteksi rintangan
* Menghindari obstacle

## ⚙️ Komponen

* Arduino Uno
* Sensor Ultrasonik
* Motor Driver L298N
* Motor DC
* Servo
* Sensor IR
